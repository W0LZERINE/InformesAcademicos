#Proyecto personal - Verano 2026

## InformesAcademicos
Primer proyecto propio que consiste en una herramienta de escritorio basica para generar informes personalizados con gráficas del progreso del alumno a partir de un archivo Excel para automatizar el relleno de evaluaciones. Desarrollado en Python con Tkint.




## ESTADO:
En desarollo

## OBJETIVO
	Generar PDFs personalizados con gráficas de progreso desde un Excel de notas.
